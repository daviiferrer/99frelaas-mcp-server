# Command Prompts

Use these prompts when asking another AI model to operate the 99Freelas account with this skill.

## Prospecting Run

```text
Use the 99freelas-growth-operator skill.

Operate inside C:\Users\luisd\Documents\Projetos 99.
Use MCP 99Freelas tools if available.

Task:
Find recent 99Freelas projects that match my winning pattern: concrete technical pain, automation/API/VPS/n8n/WhatsApp/CRM/Python/deploy, low-to-medium risk, fast delivery, and likely ticket up to R$ 500.

For each good opportunity:
- classify as sniper, consultive, high_risk, out_of_profile, or do_not_bid
- explain why
- save/update the opportunity memory
- draft a proposal only for sniper or strong consultive projects

Do not send proposals without my approval.
```

## Draft Proposal For A Specific Project

```text
Use the 99freelas-growth-operator skill.

Analyze this 99Freelas project with the MCP tools, compare it with my winning pattern, and produce a proposal draft in my best style.

Rules:
- Short, direct, technical, low-friction.
- Diagnose before selling.
- Avoid startup/platform pitch.
- If scope is unclear, write one diagnostic question instead of forcing a full proposal.
- Save/update memory in C:\Users\luisd\Documents\Projetos 99.

Project:
[paste project id, slug, URL, or title]
```
